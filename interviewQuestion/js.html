    # 关于原生js的一些知识点

    一、customEvent函数

    • 参考地址：https://developer.mozilla.org/zh-CN/docs/Web/API/CustomEvent

    # 1.1.概念

    CustomEvent接口表述由程序出于某个目的而创建的事件；（继承于Event）

    # 1.2.使用场景

    • 当有些方法只有树表手动触发才会监听到的一些内置事件，想要用js触发也能监听到的时候就可以使用该方法

    例如： window.onpopstate事件，只会在浏览器某些行为下触发，比如：点击后退、前进按钮（或者在 JavaScript 中调用history.back()、history.forward()、history.go()方法）； 调用history.pushState()或history.replaceState()不会触发 popstate 事件；

    想要可以监听的方法：

    //封装pushState方法，让其可以触发onpushstate的监听
function eventedPushState(state, title, url) {
    var pushChangeEvent = new CustomEvent("onpushstate", {
        detail: {
            state,
            title,
            url
        }
    });
    // 派发上面的事件
    document.dispatchEvent(pushChangeEvent);
    return history.pushState(state, title, url);
}

document.addEventListener(
    "onpushstate",
    function(event) {
        console.log(event.detail);
    },
    false
);

eventedPushState({}, "", "new-slug");
